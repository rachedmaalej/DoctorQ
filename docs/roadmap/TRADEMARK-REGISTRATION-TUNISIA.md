# DoctorQ Trademark Registration in Tunisia

## Overview

This document outlines the process and requirements for registering the "DoctorQ" trademark with INNORPI (Institut National de la Normalisation et de la Propriété Industrielle) in Tunisia.

## Nice Classification for DoctorQ

For a SaaS queue management system, register under the following classes:

| Class | Coverage | Priority |
|-------|----------|----------|
| **Class 42** | SaaS, cloud-based software, software as a service | **Primary** |
| **Class 38** | SMS notifications, electronic messaging | **Secondary** |
| **Class 9** | Downloadable software/mobile apps (if PWA/app planned) | Optional |
| **Class 35** | Business management services for medical practices | Optional |

---

## Required Documents

1. Application form in **2 copies** (forms available at INNORPI)
2. **5 copies of the logo/mark** (maximum 10cm x 10cm)
3. List of goods/services in **French** with Nice classification classes
4. Payment receipt for filing fees (payment on-site at INNORPI)
5. Power of attorney (if using an agent)
6. Priority document (if claiming foreign priority)

**Note:** Applications must be filed **in-person** at INNORPI. Mail applications are not accepted.

---

## Process & Timeline

| Step | Duration |
|------|----------|
| Filing at INNORPI | In-person only |
| Formal & substantive examination | ~12 months |
| Publication in Official Bulletin | After examination |
| Opposition period | 2 months from publication |
| Registration certificate | If no opposition |
| **Total timeline** | **12-18 months** (no issues) or up to 24 months |

---

## Estimated Costs

- **Official fees:** ~380 USD (~1,200 TND) for registration
- **Prior art search:** Additional fee (recommended before filing)
- **Agent fees:** Variable, if using a trademark attorney

---

## Key Legal Points

1. **First-to-file system:** Tunisia grants rights to whoever files first, not first to use - **file early**
2. **No use requirement:** You don't need to be using the mark in Tunisia to register it
3. **10-year protection:** Renewable indefinitely
4. **Madrid Protocol member:** You can extend international registrations to Tunisia
5. **Specialized IP Court:** Since 2014, Tunisia has a dedicated IP court with trained judges

## Description des Produits et Services pour "DoctorQ"

### Classe 42 - Services informatiques (Principale)

**Libellé en français:**

> Logiciels en tant que service (SaaS) pour la gestion de files d'attente dans les établissements médicaux ; services de plateforme en tant que service (PaaS) permettant aux utilisateurs de gérer les files d'attente de patients ; conception, développement et maintenance de logiciels de gestion de files d'attente ; hébergement de plateformes sur Internet pour la gestion de rendez-vous médicaux ; services informatiques de gestion de flux de patients en temps réel ; mise à disposition de logiciels non téléchargeables pour le suivi de position dans une file d'attente ; services de développement de logiciels pour cabinets médicaux et cliniques ; programmation informatique pour applications de gestion de files d'attente ; services d'assistance technique pour logiciels de gestion de files d'attente.

## Recommandation Prioritaire

Pour une protection optimale avec un budget maîtrisé, déposer au minimum:

| Priorité | Classe | Justification |
|----------|--------|---------------|
| **1** | Classe 42 | Cœur de métier - SaaS |
| **2** | Classe 38 | Notifications SMS - différenciateur clé |
| **3** | Classe 9 | Si PWA/app mobile prévue |

## Recommended Steps

1. **Prior art search first** - Ask INNORPI to check if "DoctorQ" or similar marks exist
2. **Prepare logo in final form** - Both Arabic (if applicable) and French/English versions
3. **Draft goods/services description** in French (see above)
4. **File in-person at INNORPI** in Tunis
5. **Consider Class 9** if you plan a mobile app
